  Colors

There is a large list of background, border and font. You also have animation options on hover.

There are over 150 pre-defined colors for you to choose from. The list of colors is below.

Each color gets darker as the number gets larger. If you want an orange color, not dark or not too light, go mid level orange-9 and increase or reduce that to find the right color.

Font color

To add a font color, add the color name in class eg This is red-8 <p class="red-8">This is red-8</p>

Background color

To add a color background to a button or a <div>, add "bc-" infront of the color name. The button example below has "orange-deep-14"

bc-orange-deep-14

This text is highlighted - bc-yellow-4

Border color

To add a color border to a button, add "bo-" infront of the color name. The button example below has "bo-green-14". Add a border width class, we've added 'bsw-3' here.

bo-green-14

Hover

You can change font color, background or border color when an HTML element is hovered over.

This button changes font color, background or border color on hovering over it.

HOVER OVER ME - hover-bc-purple-9 hover-brown-8 hover-bo-black hover-tr-all hover-tr-slow

Color can also be applied to tables to make them striped. There are lots of other options for animation and coloring you will see in other sections.
